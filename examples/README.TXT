DIIRT CS-Studio examples
------------------------

WARNING: this directory is only temporary located in the opi-regression-tests. It will be moved
to a suitable plugin in the main repository.


Directory structure
-------------------

The examples are grouped into 4 main categories:
 * datasources - showcase the functionality of publish/subscribe data sources such as file, ca, pva, ...
 * formula - showcase the functionality of different formula functions
 * graphene - showcase the functionality of the graphs and how can they be used to analyze data
 * services = showcase the functionality of command/response sources, such as exec, jdbc, ...


Template
--------

In the template folder, there is an example of how the screens should look like.
It defines the standard example resolution at 1280x720.
